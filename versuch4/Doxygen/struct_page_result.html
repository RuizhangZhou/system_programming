Detailed Description

This is essentially what a page returns to the main TM function (via a reference in its parameter-list). It is not returned directly to increase automation (see below).

Field Documentation

◆ child

PageState PageResult::child

What page to call if the user chooses to descent. If there is no child page, this will be a special null-page (NOT NULL).

◆ success

bool PageResult::success

Was the page successful in displaying the requested index, or does the TM function has to continue searching for a valid argument? The rationale for this goes really deep, so please consult the documentation of os_taskManMain.
