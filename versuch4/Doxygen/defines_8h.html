◆ MAX_NUMBER_OF_PROCESSES

#define MAX_NUMBER_OF_PROCESSES   8

Maximum number of processes that can be running at the same time (may be nothing > 8). This number includes the idle proc, although it is considered a system proc. The idle proc. has always id 0. The highest ID is MAX_NUMBER_OF_PROCESSES-1.
